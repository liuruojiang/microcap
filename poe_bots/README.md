# POE Bots

- `legacy/` 保存历史版本，避免和当前正式入口混淆。
- `poe_bots/microcap_top100_poe_bot.py` 是当前唯一的 POE 机器人主脚本，用于 POE 侧部署与问答入口。
- 微盘股对冲策略主脚本与 POE 机器人主脚本是两个独立入口，不混用；策略主脚本位于仓库根目录的策略脚本体系中，POE 机器人只负责对外查询与交互封装。
- 当前 POE 主脚本会自行在线拉取指数、全市场快照和个股历史，不依赖已归档的旧版单文件脚本。
- 支持命令: `信号`、`实时信号`、`成分股`、`进出名单`、`实时进出名单`、`表现`。
- 默认版本是 `1.0` 主版本；`1.1` 和 `1.2` 是备选策略版。
- 命令输出统一写入 `outputs/poe_command_outputs/`。
