# POE Bot Performance NL Design

**Goal**

增强 `poe_bots/microcap_top100_poe_bot.py` 的表现类自然语言识别，让用户在 POE 中询问某段时间的表现时，稳定返回该区间的净值曲线图、年化收益率和最大回撤；若未指定区间，则默认最近一年。

**Scope**

- 保持 POE 机器人纯 `v1.0`。
- 扩展表现类查询识别和日期区间解析。
- 默认 `表现` 查询区间改为最近一年。
- 保留现有图表、CSV 附件和摘要输出结构。

**Non-Goals**

- 不修改终端侧 `1.1 / 1.2` 查询。
- 不重写表现图绘制逻辑。
- 不引入额外数据源。

**Design**

- 在 POE 单文件中新增“表现查询识别”辅助函数，专门处理自然语言触发。
- 将默认时间窗口从“全样本”改为“最近一年”。
- 对时间短语扩展支持：
  - 近/最近/过去 `N` 年、月、周、天
  - `今年以来`、`本年以来`
  - `本月`、`上个月`
  - 保留现有 `YYYY至今`、`YYYY-YYYY`、`YYYY-MM` 等解析
- 输出仍统一包含：
  - 区间起止日期
  - 年化收益率
  - 最大回撤
  - 净值曲线 PNG

**Testing**

- `表现` 默认落到最近一年。
- `最近三个月表现`、`今年以来表现`、`上个月回撤` 等自然语言能解析出正确区间。
- `净值走势`、`年化收益率`、`最大回撤` 等表达能正确落到表现查询。
