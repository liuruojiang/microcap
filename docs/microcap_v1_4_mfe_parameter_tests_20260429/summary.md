# 微盘股 v1.4 MFE 与参数测试记录（2026-04-29）

## 结论

今天测试分三段：错过机会 MFE 诊断、现金期再进场触发、v1.4 风控参数扫描。

结论是：再进场触发的改善太弱，不值得为了小幅变化增加策略复杂度；参数扫描里有一个相对稳健的候选，但也只适合作为后续观察/复核，不建议今天直接替换正式 v1.4 默认参数。

## MFE 错过机会诊断

v1.4 现金段统计：

- 现金段数量：71
- median missed MFE：0.65%
- p90 missed MFE：3.14%
- max missed MFE：6.94%
- 最终为正的现金段：56.3%
- 先涨后回落段：16.9%

解读：v1.4 确实有少量现金期错过反弹，但大部分机会幅度不大，且不稳定。单独靠 MFE 结果不足以支持新增再进场逻辑。

## 再进场触发测试

测试逻辑：仅在 v1.4 防守无仓日观察收盘信号，次日开盘恢复部分暴露，并计入 `0.30% * scale` 入场成本。

最好看的可用组合约为：

- `rebound_2%`
- `momentum_gap >= -1%`
- 恢复 `0.6x`

但改善幅度很弱，近期窗口也不够扎实。基于今天结果，判断为不值得实盘规则复杂化。

## 操作判断

不建议因为今天的结果马上改正式 v1.4 默认参数。原因：

- 再进场触发改善太微弱，复杂度不划算。
- 参数扫描的稳健候选虽然比当前略好，但提升主要是小幅边际改善，不是策略级跃迁。
- 正式 v1.4 当前参数已经有既有文档和一致性校验，替换默认值应等下一轮更宽窗口或组合层面验证。
