# Top100 v1.4 v1.5 研究定版归档

归档日期: 2026-04-19

## 1. 本次定版结论

- `v1.4` 正式定版为:
  - 基线: `v1.1`
  - 规则: `momentum_gap` 相对 trade peak 衰减到 `25%` 时清仓
  - 恢复: 回到 peak 的 `35%` 时恢复满仓
  - 附加约束: 恢复后必须先创出新的 `gap_peak`，才允许下一次再触发

- `v1.5` 正式定版为:
  - 基线: `v1.2`
  - NAV 风控: 保持 `v1.2` 现有两级 throttle
    - `dd_moderate = 4%`
    - `dd_severe = 8%`
    - `scale_moderate = 0.85`
    - `scale_severe = 0.70`
    - `recover_dd = 3%`
  - 再叠加 signal-quality overlay:
    - `decay = 30%`
    - `scale = 0%`
    - `recover = 40%`
    - 恢复后必须先创出新的 `gap_peak`，才允许下一次再触发

说明:
- `v1.4 / v1.5` 已正式写入脚本，不再只是一次性叠加回测。
- 默认主线仍然是 `v1.0`，本次只是在研究线中确认 `v1.4 / v1.5` 的正式版本。

## 5. 今天已经尝试过且暂不推荐继续深挖的止盈/退出路径

以下路径本次研究中已经做过，当前不建议作为优先方向继续扫:

1. 固定单笔亏损阈值止损
- 包括按入场净值、按单笔 peak drawdown 的硬阈值版本。
- 修正执行层 bug 后，提升并不稳定，容易只是用收益换回撤。

2. 固定乖离率止盈
- 定义: `ratio = microcap / zz1000` 相对 `MA20(ratio)` 的固定 bias 止盈。
- 修正后结果显示整体弱于当前正式候选。

3. `momentum_gap` 峰值衰减后直接退出
- 能压回撤，但收益损失明显。
- 直接退出不如“质量变差先降风险/清仓，再按规则恢复”。

4. 止损/止盈后快速重入
- 包括“连续 2 天/3 天走强就重入”等。
- 实测整体负贡献，容易放大噪音和交易频率。

## 6. 今天没有实际尝试，但后续值得研究的止盈路径

下面这些路径本次没有正式落地，可以作为以后优化 A股/美股动量轮动组合的候选：

1. 波动率状态止盈/降风险
- 不是固定盈利多少就平仓，而是当策略或价差波动率显著升高时降风险。
- 更像 volatility-managed overlay。

2. panic-state / crash-state 条件止盈
- 当市场先大跌，再高波动，再出现强反弹时，动量更容易反噬。
- 在这种状态下做条件化降风险，可能比固定阈值止盈更合理。

3. 分档止盈，而不是全平
- 例如先减到 `50%`，再视状态决定是否退出。
- 对趋势策略更友好，不容易一刀砍飞主升段。

4. breadth / 内部扩散恶化止盈
- 不只看价格或 `momentum_gap` 本身，也看内部强势标的占比、breadth 或强弱扩散是否塌掉。
- 这类更像“信号质量衰减”而非“价格涨太多”。

5. 跨市场相对过热 + 状态过滤
- 对 A股/美股轮动组合，可研究相对强弱已经极端扩张、同时波动和市场状态也共振恶化时，才触发止盈。
- 比单独固定阈值更适合跨市场组合。

6. 时间条件止盈
- 信号持续过久后若边际强度衰减，可以研究时间止盈或 aging overlay。
- 本次未测。

## 7. 给后续 A股 / 美股动量轮动组合的迁移提示

如果把今天的经验迁到 A股 / 美股动量轮动组合，优先顺序建议是:

1. 先做“信号质量恶化后的状态化降风险”
- 不优先做硬止盈。

2. 先检查执行层一致性
- 一旦空仓或降仓，收益流必须与持仓状态严格一致。

3. 先做横向与纵向鲁棒性，再谈参数定版
- 不要只看全样本。

4. 优先解释机制，再接受参数
- 特别是恢复阈值、重入条件、分档结构，要有经济解释，不只看回测最优点。
