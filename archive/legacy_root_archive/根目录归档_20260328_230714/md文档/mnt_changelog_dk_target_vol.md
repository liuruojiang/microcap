# 更新日志: ADK目标波动率调整 15% → 20%

**日期**: 2026-03-22

## 变更内容

### bot.py

**第93行** — ADK目标波动率从15%调整为20%：

```python
# 修改前
CN_DK_TARGET_VOL = 0.15

# 修改后
CN_DK_TARGET_VOL = 0.20
```

## 变更原因

通过使用原始脚本（`CombinedStrategyBase._fetch_data()` + `_run_strategies()`）对四个子策略分别进行15% vs 20%目标波动率回测对比，结论如下：

| 策略 | 原目标 | 调到20%后组合影响 | 决策 |
|:-----|:------|:-----------------|:-----|
| **Sub-A** | 15% | 收益+0.8~1.6%, 夏普持平 | 维持15% |
| **ADK** | 15% | 收益+2.7~3.6%, 夏普持平或微升 | **→ 调整为20%** |
| **Sub-B** | 20% | 已是20%, 降到15%会变差 | 维持20% |
| **Sub-C** | 15% | 收益-0.2%, 夏普-0.05, 回撤增大 | 维持15% |

| 8年 | +20.64% / 夏普2.08 / 回撤-8.59% | +24.24% / 夏普2.18 / 回撤-9.81% | 收益+3.61%, 夏普+0.10 |
| 12年 | +18.39% / 夏普1.95 / 回撤-8.59% | +21.12% / 夏普2.03 / 回撤-9.81% | 收益+2.73%, 夏普+0.08 |

## 影响范围

- **信号机器人**: ADK策略的vol-scaling会使用新的20%目标，实时信号中显示的杠杆倍数会相应提高
- **记录机器人**: 无需修改（记录机器人读取信号机器人输出，不涉及策略参数）
- **其他子策略**: 不受影响，Sub-A/Sub-B/Sub-C参数均未变动
