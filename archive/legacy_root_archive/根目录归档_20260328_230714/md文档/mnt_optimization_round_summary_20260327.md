# 2026-03-27 本轮优化结论备忘录

## 正式落地
- 正式采纳方案: `Sub-B`
- 参数调整:
  - `US_ROT_TARGET_VOL = 0.25`
  - `US_ROT_MAX_LEV = 2.0`
- 杠杆放大范围不变:
  - 仅限 `US_ROT_FUTURES` 中的三类资产
  - `QQQ -> QQQM`
  - `GLD -> GLDM`
  - `TLT -> VGLT`
- 不扩展到“全资产可加杠杆”

## 采纳原因
- 这是本轮唯一一个在可复现口径下仍然成立、并且适合进入正式主策略的调整
- 相比基线 `20% / 1.5x / 仅三类可放大`，该方案在组合层面保持正向改进
- 交易频率增幅可接受
  - `Sub-B` 总交易日频率约从 `25.79` 次/年升至 `28.66` 次/年
  - 约增加 `2.87` 次/年
  - 增幅约 `11.1%`

## 已排除方向
- `ADK` 周频确认换配对
  - 交易频率下降，但收益和 Sharpe 下滑过多
- `ADK` Top-2 配对等权
  - 交易频率反而上升，表现变差
- `ADK` 条件触发换配对
  - 不优于简单 retention 版本
- `B` 向 `C` 的简单 overlay 化
  - 不能有效降低与 `C` 的重叠，组合效果变差
- 组合层回撤后统一降风险
  - 日频版操作次数过多，排除
  - 周频版新增操作仍不低，且收益/回撤交换不够好，不进主策略
- “全资产可加杠杆 + 融资成本”
  - 在当前可复现框架下不成立

## 仅保留研究备选
- `ADK 1.05x retention`
  - 能小幅降低结构性交易次数
  - 但组合表现轻微走弱
  - 仅保留为研究备选，不进入正式主策略

## 本轮收口结论
- 本轮“脑洞大开式”策略优化到此收口
- 当前唯一正式采纳的策略调整为:
  - `Sub-B 25% / 2.0x / 仅三类可放大`
- 其余方向全部归入:
  - 排除
  - 或仅研究备选
